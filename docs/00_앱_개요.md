# BeamChaser 앱 개요

## 앱 이름
**BeamChaser** (런빔)

## 한 줄 소개
라인 레이저 페이스메이커와 연동하여 목표 페이스를 시각적으로 안내하는 러닝 앱

## 핵심 콘셉트
BeamChaser은 Arduino 기반 하드웨어 장치(레이저 페이스메이커)와 BLE(Bluetooth Low Energy)로 연결되어, 달리는 러너의 목표 페이스를 **레이저 라인**으로 지면에 투사합니다. 러너는 레이저 라인을 따라 달리며 직관적으로 자신의 페이스를 조절할 수 있습니다.

---

## 기술 스택

| 항목 | 기술 |
|------|------|
| 플랫폼 | iOS 17.0+ |
| UI 프레임워크 | SwiftUI |
| 빌드 도구 | XcodeGen (`project.yml`) |
| BLE 통신 | CoreBluetooth + HM-10 모듈 |
| 위치 추적 | CoreLocation + CMPedometer (센서 퓨전) |
| 건강 데이터 | HealthKit |
| 인증 | Sign in with Apple |
| 백엔드 | Firebase (SDK 11.6.0, 현재 비활성) |
| 지도 | MapKit |
| 하드웨어 | Arduino UNO + GPS NEO M9N + 서보 + 레이저 3개 |

## 앱 구조

### 탭 구조 (메인 TabView)
```
┌─────────────────────────────────────────┐
│                 TabView                  │
├──────────┬──────────┬──────────┬────────┤
│   홈     │   기록   │ 커뮤니티 │ 프로필 │
│ HomeView │ History  │Community │Profile │
└──────────┴──────────┴──────────┴────────┘
```

### 전체 화면 흐름
```
홈 (HomeView)
├── 장치 연결 (DeviceConnectionView)
│   ├── BLE 스캔 & 연결
│   ├── 서보 각도 조절
│   └── 레이저 테스트
├── 러닝 설정 (PaceSetupView)
│   ├── 일반 모드 (거리/시간/페이스 목표)
│   └── 인터벌 모드 (다구간 페이스)
└── 러닝 중 (RunActiveView)
    ├── 실시간 지도 (RunMapView)
    ├── 페이스 상태 (앞서감/유지/뒤처짐)
    ├── 레이저 갭 표시
    └── 완료 요약

기록 (RunHistoryView)
├── 전체 통계 카드
├── 페이스 추이 그래프
└── 상세 보기 (RunDetailView)
    ├── 경로 지도
    ├── 구간별 스플릿
    ├── 목표 달성 여부
    └── 공유 카드 (RunShareCardView)

커뮤니티 (CommunityView)
├── 러닝 메이트 모집
│   ├── 지도에 모집 위치 표시
│   └── 모집글 작성
└── 피드
    ├── 러닝 결과 공유
    └── 자유 게시판

프로필 (ProfileView)
├── Apple 로그인
├── 러너 등급 (6단계)
├── 월간 목표 링
├── 뱃지 컬렉션 (16종)
├── 누적 통계
└── 설정 (SettingsView)
    ├── 장치 설정
    ├── 레이저 캘리브레이션
    ├── 러닝 설정
    ├── 음성 안내
    ├── 건강 & 운동 (HealthKit)
    └── 화면 모드 (다크/라이트/시스템)
```

## 핵심 서비스

| 서비스 | 파일 | 역할 |
|--------|------|------|
| BLEService | `Services/BLEService.swift` | Arduino 장치 BLE 통신 |
| LocationService | `Services/LocationService.swift` | GPS + 보수계 센서 퓨전 위치 추적 |
| RunSessionManager | `Services/RunSessionManager.swift` | 러닝 세션 라이프사이클 관리 |
| PaceMakerEngine | `Services/PaceMakerEngine.swift` | 목표 페이스 대비 갭 계산 |
| HealthKitService | `Services/HealthKitService.swift` | Apple 건강 앱 연동 |
| AuthService | `Services/AuthService.swift` | Sign in with Apple 인증 |
| ProfileService | `Models/RunnerProfile.swift` | 등급/뱃지/월간 목표 관리 |

## 디자인 시스템

| 요소 | 설명 |
|------|------|
| 메인 컬러 | `RBColor.accent` = Orange |
| 배경 | 다크: 블랙 / 라이트: 연회색 |
| 카드 배경 | 다크: 0.11 화이트 / 라이트: 화이트 |
| 폰트 | `RBFont.hero` (둥근 블랙), `RBFont.metric` (모노스페이스 헤비), `RBFont.label` (둥근 세미볼드) |
| 레이저 도트 | `LaserDot` — 빨간 글로우 원형 |
| 성공/위험 | 녹색 `RBColor.success` / 빨강 `RBColor.danger` |
| 외관 모드 | 시스템 / 라이트 / 다크 3가지 지원 |
