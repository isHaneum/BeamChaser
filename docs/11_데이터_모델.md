## 1. 러닝 기록 (RunRecord)

```swift
struct RunRecord: Codable, Identifiable {
    let id: UUID
    let date: Date
    let distance: Double          // 총 거리 (m)
    let duration: TimeInterval    // 총 시간 (초)
    let averagePace: Double       // 평균 페이스 (분/km)
    let calories: Double          // 소모 칼로리
    let cadence: Int              // 평균 케이던스 (spm)
    let heartRate: Int            // 평균 심박수
    let routePoints: [RoutePoint] // 경로 좌표 배열
    let splitPaces: [Double]      // km별 스플릿 페이스
    let goal: RunGoal?            // 설정한 목표 (있을 경우)
    let goalAchieved: Bool        // 목표 달성 여부
    let runMode: RunMode          // 러닝 모드
    let intervals: [IntervalSegment]? // 인터벌 구간 (인터벌 모드)
}
```

### 저장 방식
- `UserDefaults`에 `"savedRunRecords"` 키로 JSON 인코딩 저장
- 앱 시작 시 `JSONDecoder`로 로드

## 5. 인터벌 구간 (IntervalSegment)

```swift
struct IntervalSegment: Codable, Identifiable {
    let id: UUID
    let type: IntervalType       // 러닝 / 휴식
    let duration: TimeInterval   // 구간 시간 (초)
    let targetPace: Double?      // 구간 목표 페이스
}

enum IntervalType: String, Codable {
    case run   // 달리기 구간
    case rest  // 휴식 구간
}
```

### 프리셋 인터벌 세트
| 이름 | 구성 |
|------|------|
| 초보자 | (달리기 1분 + 걷기 2분) × 5 |
| 중급자 | (달리기 3분 + 걷기 1분) × 6 |
| 상급자 | (달리기 5분 + 걷기 1분) × 5 |
| 스프린트 | (전력질주 30초 + 휴식 90초) × 8 |

## 8. 러너 프로필 (RunnerProfile)

```swift
class RunnerProfile: ObservableObject {
    @Published var nickname: String
    @Published var totalDistance: Double      // 누적 거리 (m)
    @Published var totalRuns: Int             // 총 러닝 횟수
    @Published var totalTime: TimeInterval    // 총 러닝 시간
    @Published var bestPace: Double           // 최고 페이스 (분/km)
    @Published var currentStreak: Int         // 연속 러닝 일수
    @Published var level: RunnerLevel         // 러너 레벨
    @Published var badges: [Badge]            // 획득한 뱃지
    @Published var monthlyGoal: MonthlyGoal   // 월간 목표
    @Published var strideLength: Double       // 보폭 (m, 만보기 보정용)
    @Published var weight: Double             // 체중 (kg, 칼로리 계산용)
}
```

### 저장 방식
- `UserDefaults`에 각 필드를 개별 키로 저장
- 앱 시작 시 자동 로드

## 11. 월간 목표 (MonthlyGoal)

```swift
struct MonthlyGoal: Codable {
    var targetDistance: Double    // 목표 거리 (km)
    var targetRuns: Int          // 목표 러닝 횟수
    var targetTime: TimeInterval // 목표 러닝 시간 (초)
    var currentDistance: Double   // 현재 거리
    var currentRuns: Int         // 현재 횟수
    var currentTime: TimeInterval // 현재 시간
    var month: Int               // 대상 월
    var year: Int                // 대상 연도
}
```

프로필 화면에서 3개의 링(거리/횟수/시간)으로 진행 상황을 시각화합니다.

## 12. 레이저 캘리브레이션 (LaserCalibration)

```swift
struct LaserCalibration {
    var heightCm: Double         // 장착 높이 (cm)
    var projectionDistanceM: Double // 투사 거리 (m)
    var servoAngle: Int          // 서보 각도 (60~110°)
}
```

### 투사 거리 공식
```
투사 거리(m) = 장착 높이(cm) × tan(서보 각도) / 100
```

설정 화면에서 슬라이더로 서보 각도를 조절하면 실시간으로 투사 거리가 계산됩니다.

## 13. BLE 관련 모델

### DeviceStatus (장치 상태)
```swift
struct DeviceStatus {
    let battery: Int        // 배터리 (0~100%)
    let isLaserOn: Bool     // 레이저 활성 여부
    let servoAngle: Int     // 서보 각도 (0~180°)
    let zone: DeviceZone    // 현재 Zone
    let pace: Int           // 현재 페이스 (초/km)
    let distance: Int       // 누적 거리 (m)
}
```

### DeviceZone (Zone 열거형)
```swift
enum DeviceZone: UInt8 {
    case none  = 0  // 없음
    case blue  = 1  // 빠름
    case green = 2  // 적정
    case red   = 3  // 느림
}
```

### LaserCommand (BLE 명령)
```swift
enum LaserCommand {
    case laserOff           // 0x00
    case laserOn            // 0x01
    case setPace(Int)       // 0x02 + 2바이트
    case setAngle(Int)      // 0x03 + 1바이트
    case setZone(DeviceZone)// 0x04 + 1바이트
    case startRun           // 0x05
    case stopRun            // 0x06
    case requestStatus      // 0x07
}
```
