# 프로필 화면 (ProfileView)

## 파일 위치
`RunBeam/Views/Profile/ProfileView.swift`

## 화면 설명
네 번째 탭으로 표시되는 프로필 화면입니다. 러너 등급, 뱃지 컬렉션, 월간 목표, 누적 통계를 확인할 수 있습니다.

---

## 네비게이션 제목
"프로필" (large 스타일)

## 네비게이션 바
- **우측**: 설정 기어 아이콘 → `SettingsView`로 이동

---

## 화면 구성

### 1. 로그인 섹션 (미로그인 시)
```
┌──────────────────────────────────────┐
│           👤 (80pt 원형)             │
│                                      │
│    로그인하고 기록을 저장하세요       │
│                                      │
│  ┌─────────────────────────────┐     │
│  │  Sign in with Apple         │     │
│  └─────────────────────────────┘     │
│                                      │
│  (시뮬레이터: 테스트 로그인 버튼)     │
└──────────────────────────────────────┘
```
- Sign in with Apple 버튼 (흰색 스타일)
- 로그인 성공 시 이름/이메일 저장
- 에러 메시지 표시 (빨간 텍스트)

### 2. 프로필 헤더 (로그인 시)
```
┌──────────────────────────────────────┐
│         ┌────────┐                   │
│         │  X     │  ← 이니셜 (accent │
│         │        │     그라데이션)    │
│         └────────┘                   │
│          사용자 이름                  │
│        user@email.com                │
│         🔥 3일 연속                   │
└──────────────────────────────────────┘
```
- 원형 아바타: 이름 첫 글자, accent 그라데이션 배경
- 이메일 (있으면)
- 연속 러닝 스트릭 배지 (1일 이상)

### 3. 러너 등급 카드
```
┌──────────────────────────────────────┐
│ 러너 등급                            │
│ 실버                         🥈     │
│                              (36pt) │
│ 12.5 km ━━━━━●━━━━━━━ 50.0 km      │
│ ████████████░░░░░░░░░░░░░           │
│ 다음 등급: 골드                      │
└──────────────────────────────────────┘
```

#### 등급 시스템 (6단계)
| 등급 | 누적 거리 | 아이콘 | 색상 |
|------|----------|--------|------|
| 스타터 | 0 km | `figure.walk` | 회색 |
| 브론즈 | 10 km | `medal` | 갈색 |
| 실버 | 50 km | `medal.fill` | 은색 |
| 골드 | 100 km | `trophy` | 금색 |
| 레이저 | 300 km | `laser.burst` | 빨강 |
| 빔 마스터 | 1000 km | `bolt.shield.fill` | 오렌지 |

- 진행 바: 현재 거리 / 다음 등급 거리
- 최고 등급 달성 시: "최고 등급 달성! 🏆"

### 4. 월간 목표 카드
```
┌──────────────────────────────────────┐
│ 이번 달 목표                    4월  │
│                                      │
│   ┌──────┐        ┌──────┐          │
│   │ 75%  │        │ 60%  │          │
│   │  ○   │        │  ○   │          │
│   └──────┘        └──────┘          │
│    러닝              거리            │
│   9/12회          30.0/50.0km       │
└──────────────────────────────────────┘
```
- 원형 프로그레스 링 (러닝 횟수, 거리)
- 러닝 횟수: 오렌지 링 (기본 목표: 12회/월)
- 거리: 초록 링 (기본 목표: 50km/월)
- 중앙에 퍼센트 표시

### 5. 뱃지 컬렉션 (4열 그리드)
```
┌──────────────────────────────────────┐
│ 뱃지 컬렉션                  5/16   │
│                                      │
│  🟠👟   🟠5️⃣   ⬜🔟   ⬜50   │
│  첫발걸음  꾸준한   10회   하프센추리│
│                                      │
│  🟠🏁   ⬜🏁   ⬜🏆  ⬜🏆   │
│  5K완주   10K    하프   풀마라톤 │
│                                      │
│  🟠🎯   🟠🔥   ⬜🔥  ⬜⚡   │
│  목표달성  3일    7일    30일   │
│                                      │
│  ⬜🌙   ⬜🌅   ⬜🐇  ⬜💫   │
│  야간     얼리   스피드  레이저 │
└──────────────────────────────────────┘
```

#### 뱃지 종류 (16종)
| 뱃지 | 조건 | 아이콘 |
|------|------|--------|
| 첫 발걸음 | 1회 러닝 | `shoe.fill` |
| 꾸준한 러너 | 5회 러닝 | `5.circle.fill` |
| 10회 달성 | 10회 러닝 | `10.circle.fill` |
| 하프 센추리 | 50회 러닝 | `50.circle.fill` |
| 5K 완주 | 5km 1회 달리기 | `flag.fill` |
| 10K 완주 | 10km 1회 달리기 | `flag.checkered` |
| 하프마라톤 | 21.1km 1회 | `trophy` |
| 풀마라톤 | 42.195km 1회 | `trophy.fill` |
| 목표 달성 | 목표 페이스 달성 | `target` |
| 3일 연속 | 3일 연속 러닝 | `flame` |
| 7일 연속 | 7일 연속 러닝 | `flame.fill` |
| 30일 연속 | 30일 연속 러닝 | `bolt.fill` |
| 야간 러너 | 밤 9시 이후 러닝 | `moon.fill` |
| 얼리버드 | 아침 6시 전 러닝 | `sunrise.fill` |
| 스피드 데몬 | 4분/km 이하 | `hare.fill` |
| 레이저 동기화 | 전 구간 ±5m 이내 | `laser.burst` |

- 획득: 오렌지 배경 + 밝은 아이콘
- 미획득: 어두운 배경 + 흐린 아이콘

### 6. 누적 통계
```
┌──────────────────────────────────────┐
│ 누적 통계                            │
│  총 러닝    총 거리     총 시간      │
│   15회      62.3km    5:32:10      │
├──────────────────────────────────────┤
│  평균 페이스  최고 페이스  연속 일수  │
│   5'14"      4'50"       3일       │
├──────────────────────────────────────┤
│         🚪 로그아웃                  │
└──────────────────────────────────────┘
```

## HealthKit 연동
- `onAppear`에서 뱃지/등급 평가 (`evaluateAfterRun`)
- HealthKit에서 키 정보 자동 가져오기 (`userHeightCm`)
- 기본 키(170cm) 또는 미설정 시에만 덮어씌움

---

## 의존하는 서비스
| 서비스 | 용도 |
|--------|------|
| `ProfileService` | 등급, 뱃지, 스트릭, 월간 목표 |
| `RunSessionManager` | 저장된 기록 조회 |
| `AuthService` | Apple 로그인, 사용자 정보 |
| `HealthKitService` | 키 정보 자동 가져오기 |
