# 장치 연결 화면 (DeviceConnectionView)

## 파일 위치
`BeamChaser/Views/Home/DeviceConnectionView.swift`

## 화면 설명
Arduino BeamChaser 장치와 BLE(HM-10 모듈)로 연결하고, 서보 각도 조절 및 레이저 테스트를 할 수 있는 화면입니다.

---

## 화면 구성

### 1. 네비게이션 바
- **제목**: "장치 연결" (인라인)
- **우측 버튼**:
  - 연결 시: "연결 해제" (빨간 텍스트)
  - 미연결 시: 메뉴 버튼
    - "BeamChaser 검색" — FFE0 서비스 UUID 필터로 스캔
    - "모든 BLE 장치 검색" — 전체 BLE 스캔

### 2. 연결됨 카드 (연결 시에만 표시)
```
┌──────────────────────────────────────┐
│ ✅ BeamChaser              배터리 85%  │
│    연결됨                ● 적정     │
│                         레이저 ON   │
└──────────────────────────────────────┘
```
- 초록 체크마크 아이콘
- 장치 이름 표시
- 우측에 배터리 잔량, Zone 상태 (색상 원), 레이저 ON/OFF 표시
- 초록 테두리 + 연한 초록 배경

### 3. 서보 제어 카드 (연결 시에만 표시)
```
┌──────────────────────────────────────┐
│ ∠ 서보 각도                    85°  │
│ ━━━━━━━━●━━━━━━━━━━━━ (슬라이더)   │
│ [60°] [75°] [85°] [95°] [110°]     │
└──────────────────────────────────────┘
```
- 슬라이더 범위: 60° ~ 110° (1° 단위)
- 프리셋 버튼 5개: 60°, 75°, 85°, 95°, 110°
- 값 변경 시 `bleService.setServoAngle()` 호출하여 Arduino에 즉시 전송
- 현재 선택된 프리셋은 하이라이트 처리 (accent 배경)

### 4. 레이저 테스트 카드 (연결 시에만 표시)
```
┌──────────────────────────────────────┐
│ 💡 레이저 테스트                     │
│ [파랑] [초록] [빨강]                │
│ [레이저 ON]  [레이저 OFF]           │
└──────────────────────────────────────┘
```
- **Zone 버튼**: 파랑(빠름), 초록(적정), 빨강(느림) 각각의 색상으로 표시
- **레이저 ON/OFF**: 전체 레이저 켜기/끄기

### 5. 에러 메시지
- 연결 오류 발생 시 빨간 배경으로 에러 텍스트 표시
- 삼각 느낌표 아이콘 + 에러 내용

### 6. 발견된 장치 목록
```
┌──────────────────────────────────────┐
│ 검색된 장치               🔄(로딩)  │
├──────────────────────────────────────┤
│ 📡 BeamChaser-001                      │
│    A8B3C4D5...                      │
├──────────────────────────────────────┤
│ 📡 HM-10                            │
│    F1E2D3C4...                      │
└──────────────────────────────────────┘
```
- 장치가 없으면: 안테나 아이콘 + "장치를 검색해주세요" + "HM-10 BLE 모듈이 켜져 있는지 확인하세요"
- 스캔 중이면 우측에 ProgressView 로딩 표시
- 장치 탭 시 `bleService.connect(to:)` 호출

## BLE 통신 프로토콜

### 사용하는 UUID
| 항목 | UUID |
|------|------|
| Service | `0000FFE0-0000-1000-8000-00805F9B34FB` |
| Characteristic | `0000FFE1-0000-1000-8000-00805F9B34FB` |

### 명령 전송 (앱 → 장치)
| 명령 | 바이트 | 설명 |
|------|--------|------|
| 레이저 OFF | `0x00` | 레이저 끄기 |
| 레이저 ON | `0x01` | 레이저 켜기 |
| 페이스 설정 | `0x02 HH LL` | 목표 페이스 (초/km, big-endian) |
| 각도 설정 | `0x03 AA` | 서보 각도 (0~180°) |
| Zone 설정 | `0x04 ZZ` | 0=없음, 1=파랑, 2=초록, 3=빨강 |
| 러닝 시작 | `0x05` | GPS 트래킹 + 레이저 활성화 |
| 러닝 종료 | `0x06` | 레이저 끄기 + 대기 복귀 |
| 상태 요청 | `0x07` | 즉시 상태 패킷 요청 |

### 상태 수신 (장치 → 앱)
10바이트 패킷: `STX(0xAA) battery laserOn servoAngle zone pace_H pace_L dist_H dist_L ETX(0x55)`
