# 러닝 기록 (RunHistoryView + RunDetailView)

## 파일 위치
- `BeamChaser/Views/History/RunHistoryView.swift` — 기록 목록
- `BeamChaser/Views/History/RunDetailView.swift` — 기록 상세

---

## 1. 기록 목록 화면 (RunHistoryView)

### 화면 설명
두 번째 탭으로 표시되는 러닝 기록 목록입니다. 전체 통계, 페이스 추이 그래프, 개별 기록 카드가 표시됩니다.

### 네비게이션 제목
"러닝 기록" (large 스타일)

### 화면 구성

#### 빈 상태 (기록 없음)
```
┌────────────────────────────────┐
│       🏃‍♂️ (56pt 아이콘)        │
│    러닝 기록이 없습니다         │
│    첫 러닝을 시작해보세요!      │
└────────────────────────────────┘
```

#### 전체 통계 카드
```
┌──────────────────────────────────────┐
│ 전체 통계                    5회 러닝│
│                                      │
│  총 거리    총 시간    평균 페이스  최고│
│  12.5      1:05:30    5'14"     4'50"│
│   km                   /km       /km │
└──────────────────────────────────────┘
```
- 100m 미만 기록은 통계에서 제외
- 최고 페이스: 가장 낮은(빠른) 평균 페이스 → 초록색

#### 페이스 추이 카드 (2개 이상 기록 시)
```
┌──────────────────────────────────────┐
│ 페이스 추이             ↗ 향상 중    │
│                                      │
│     ●                                │
│       \    ●                         │
│         \  / \   ●                   │
│          ●     \/                    │
│                                      │
│ 처음: 5'40"              최근: 5'10" │
└──────────────────────────────────────┘
```
- 최근 10회 기록의 페이스 추이를 미니 라인 차트로 표시
- 향상 중(마지막 < 처음): 초록 + ↗
- 하락 중(마지막 > 처음): 빨강 + ↘
- 그라데이션 라인 (accent → success/danger)
- 데이터 포인트 (최근: 6pt accent / 나머지: 4pt 반투명)

#### 기록 리스트
```
┌──────────────────────────────────────┐
│ MAR │ │  2.45 km                 🔴  │
│  28 │ │  🏎 5'02"/km  ⏱ 12:34   >  │
└──────────────────────────────────────┘
```
- 좌측: 날짜 뱃지 (월 약어 + 일)
- 중앙: 거리(20pt 메트릭), 페이스 + 시간 (라벨, 12pt)
- 우측: 목표 페이스 있으면 레이저 도트(🔴), 쉐브론(>)
- 탭 시 `RunDetailView`로 이동

## 2. 기록 상세 화면 (RunDetailView)

### 화면 설명
개별 러닝 기록의 상세 정보를 표시합니다.

### 네비게이션
- **제목**: "러닝 상세" (인라인)
- **우측 버튼**: 공유 아이콘 → 공유 카드 시트 표시

### 화면 구성

#### 경로 지도 (2개 이상 좌표)
```
┌──────────────────────────────────────┐
│      ┌─────────────────┐            │
│      │     지도         │   240pt   │
│      │  🟢━━━━━━🔴     │   높이    │
│      └─────────────────┘            │
└──────────────────────────────────────┘
```
- 시작점: 초록 원
- 종료점: 빨간 원
- 경로: 오렌지 그라데이션 라인 (4pt)
- 모서리: 20pt 라운드

#### 날짜
- `date: .abbreviated, time: .shortened` 형식

#### 메인 수치
```
┌──────────────────────────────────────┐
│   거리      │   시간    │  평균 페이스│
│   2.45 km  │  12:34   │   5'02"    │
└──────────────────────────────────────┘
```

#### 추가 상세 수치
```
┌──────────────────────────────────────┐
│  평균 속도:  11.8 km/h              │
│  최고 속도:  15.2 km/h              │
├──────────────────────────────────────┤
│  총 고도 상승: 45 m                 │
│  칼로리 (추정): 350 kcal            │
└──────────────────────────────────────┘
```
- 칼로리: MET 10 × 70kg × 시간 (추정치)
- 최고 속도: routePoints의 speed 최대값 × 3.6

#### 목표 페이스 결과 (목표 있을 때)
```
┌──────────────────────────────────────┐
│ 🔴 목표 페이스              달성!    │
│    5'30"/km               (초록 배지)│
│                           또는       │
│                           +15초 미달성│
│                           (빨강 배지)│
└──────────────────────────────────────┘
```

#### 구간별 페이스 (km 스플릿)
```
┌──────────────────────────────────────┐
│ 구간별 페이스                        │
│                                      │
│ 1 km  ███████████████████  5'10"    │
│ 2 km  ████████████████     5'25"    │
│ 3 km  █████████████████    5'18"    │
└──────────────────────────────────────┘
```
- GPS 경로 데이터에서 1km마다 구간 시간 계산
- 가장 빠른 구간: 초록 바
- 나머지 구간: 연한 오렌지 바
- 바 길이: 최고 페이스 기준 비율

#### 기록 분석 (이전 기록 대비)
```
┌──────────────────────────────────────┐
│ 기록 분석                            │
│ ↗ 페이스 향상!                  평균 │
│ 평균 대비 15초/km 빠름         5'30" │
└──────────────────────────────────────┘
```
- 이전 전체 기록의 평균 페이스와 비교
- 향상: 초록 + ↗ / 하락: 빨강 + ↘
- 이전 기록이 없으면 미표시

## 3. 러닝 공유 카드 (RunShareCardView)

### 파일 위치
`BeamChaser/Views/Profile/RunShareCardView.swift`

실제 화면 구현:

`BeamChaser/Views/Profile/RunShareComposerScene.swift`

### 화면 설명
러닝 기록을 시각적 카드로 생성하고, 2단계 편집 흐름으로 공유하는 전체 화면입니다.

- 1단계: 공유 화면
- 고정 미리보기 카드 표시
- 수치 / 스타일 / 사진 탭으로 카드 구성 조정
- 완료 시 현재 카드 이미지를 커뮤니티 피드용 이미지로 업로드

- 2단계: 배경 편집 화면
- 템플릿 선택
- 넓은 미리보기 카드 리스트 확인
- 화이트 / 사진 배경 / 오버레이 / 투명 배경 선택
- 사용자가 직접 앨범 사진을 넣고 즉시 카드 배경으로 반영

### 카드 디자인
```
┌──────────────────────────────────────┐
│ ● BEAMCHASER                         │
│                                      │
│      주행 경로 라인 오버레이         │
│                                      │
│  5.24 km                             │  ← 대형 hero 수치
│                                      │
│  시간   페이스   칼로리              │  ← 하단 보조 수치
└──────────────────────────────────────┘
```
- 화이트 / 사진 / 오버레이 / 투명 카드 배경 지원
- 경로 라인은 오렌지-레드 계열의 레이저 스타일을 기본 사용
- 거리, 시간, 페이스, 칼로리 노출 순서를 드래그로 변경 가능

### 기능
- **공유하기**: `UIActivityViewController`로 이미지 공유
- **배경 사진 선택**: `PhotosPicker`로 사용자가 직접 사진 추가
- **피드 등록**: 렌더된 카드 이미지를 `Firebase Storage`의 `feed_photos/{uid}/share_cards/...` 경로에 업로드 후 `feedPosts.photoURL`에 저장
- 3x 해상도로 렌더링 (`ImageRenderer`, scale: 3.0)
