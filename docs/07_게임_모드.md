# 게임 모드 (GameSelectView)

## 파일 위치
`RunBeam/Views/Game/GameSelectView.swift`

## 화면 설명
RunBeam 레이저를 활용한 다양한 러닝 게임 모드를 선택하는 화면입니다. 현재는 **모든 게임이 "준비 중" 상태**이며, 향후 업데이트에서 구현될 예정입니다.

---

## 진입 경로
홈 화면 → (향후 추가 예정) 또는 직접 네비게이션

## 네비게이션 제목
"게임 모드" (인라인)

---

## 게임 타입 (4종)

| 게임 | 아이콘 | 색상 | 설명 |
|------|--------|------|------|
| 하트런 | `heart.fill` | 빨강 | 하트 모양 경로를 따라 달리며 레이저로 하트를 그려보세요 |
| 사과런 | `apple.logo` | 초록 | 지도 위 사과를 수집하며 달리는 수집형 러닝 |
| HI-FIVE! | `hand.raised.fill` | 노랑 | 다른 러너와 만나면 하이파이브! 소셜 러닝 게임 |
| ROAD RUN | `road.lanes` | 파랑 | 지도 땅따먹기! 달린 도로를 점령하세요 |

---

## 게임 카드 디자인
- 좌측: 52×52 아이콘 박스 (각 게임 색상 12% 배경)
- 중앙: 게임 이름 (16pt) + "준비 중" 캡슐 배지 + 설명 (12pt, 2줄 제한)
- 우측: 쉐브론(>)
- 카드 배경: `RBColor.cardBg`, 16pt 라운드

---


## 현재 상태
- 모든 게임은 UI만 존재하며 실제 게임 로직은 미구현
- 탭 시 플레이스홀더 화면으로 이동
- 향후 업데이트에서 레이저 하드웨어와 연동한 게임 기능 추가 예정
