# 설정 화면 (SettingsView)

## 파일 위치
`BeamChaser/Views/Settings/SettingsView.swift`

## 화면 설명
프로필 탭 우측 상단 기어 아이콘에서 진입하는 설정 화면입니다. 장치, 레이저 캘리브레이션, 러닝, 음성 안내, 건강 연동, 화면 모드 등 전체 앱 설정을 관리합니다.

---

## 화면 구성 (섹션별)

### 1. BeamChaser 장치 섹션

#### 장치 연결
```
┌──────────────────────────────────────┐
│ 📡 연결됨: BeamChaser-001          >   │
├──────────────────────────────────────┤
│ 🔋 배터리               ████ 85%   │
│                          (충전 중)   │
├──────────────────────────────────────┤
│ ☀️ 레이저 밝기              80%     │
│    ━━━━━━━━●━━━━ (슬라이더)         │
└──────────────────────────────────────┘
```
- 장치 연결 행: `DeviceConnectionView`로 NavigationLink
- 연결 시 초록 아이콘 + 장치 이름 / 미연결 시 오렌지
- 배터리 행: 비주얼 바 + 퍼센트 + 충전 상태
- 레이저 밝기 슬라이더: 10% ~ 100% (10% 단위)
- `@AppStorage("laserBrightness")` 저장

### 2. 레이저 캘리브레이션 섹션

```
┌──────────────────────────────────────┐
│ 🧍 키                      170 cm   │
│                          [- ] [+]   │
├──────────────────────────────────────┤
│ 👕 장착 위치    [가슴|허리|팔]       │
│                 (Segmented Picker)    │
├──────────────────────────────────────┤
│ ∠ 레이저 각도 미세조절      +0.0°    │
│   ━━━━━━━━━●━━━━━ (-5° ~ +5°)      │
├──────────────────────────────────────┤
│ 📏 예상 투사 거리          약 2.1m   │
├──────────────────────────────────────┤
│ 🔄 기기에 캘리브레이션 전송          │ ← 연결 시에만 표시
└──────────────────────────────────────┘
```

#### 캘리브레이션 계산
- 장착 높이 = 키(cm) / 100 × 장착 위치 비율
  - 가슴: 0.75
  - 허리: 0.55
  - 팔: 0.65
- 투사 거리 = 장착 높이 / tan((30 + 각도 오프셋)° × π / 180)
- "기기에 캘리브레이션 전송" → `bleService.setServoAngle(85 + offset)`

#### 저장 키
| 키 | 기본값 | 설명 |
|----|--------|------|
| `userHeightCm` | 170 | 사용자 키 (cm) |
| `mountPosition` | "가슴" | 장착 위치 |
| `laserAngleOffset` | 0.0 | 각도 보정 (-5~+5°) |

### 3. 러닝 섹션

```
┌──────────────────────────────────────┐
│ ⏸ 자동 일시정지            [토글]   │
├──────────────────────────────────────┤
│ 📳 햅틱 피드백              [토글]   │
└──────────────────────────────────────┘
```
- 자동 일시정지: `@AppStorage("autoPause")`
- 햅틱 피드백: `@AppStorage("hapticFeedback")`

### 4. 음성 안내 섹션

```
┌──────────────────────────────────────┐
│ 🔊 음성 안내                [토글]   │
├──────────────────────────────────────┤
│  (음성 안내 활성 시 아래 추가 옵션)   │
│                                      │
│ 📏 거리별 안내 주기         1.0km    │
│    ━━━━●━━━━━━ (0~5km, 0.5단위)    │
│    설정 거리마다 페이스, 거리, 시간 안내│
├──────────────────────────────────────┤
│ ⚠️ 페이스 이탈 경고         15m      │
│    ━━━━●━━━━━━ (5~50m, 5단위)      │
│    레이저와 설정 거리 이상 벌어지면 경고│
├──────────────────────────────────────┤
│ ⏱ 카운트다운 음성           [토글]   │
└──────────────────────────────────────┘
```
- 거리 안내 주기: 0이면 끔
- 페이스 이탈 경고: 레이저 갭이 설정 거리 이상이면 음성 경고
- 카운트다운 음성: 시작 카운트다운 음성 안내

### 5. 건강 & 운동 섹션

```
┌──────────────────────────────────────┐
│ ❤️ Apple 건강 앱 연동       연동됨   │
│                          (또는 권한 요청)│
├──────────────────────────────────────┤
│ 🏃 운동 자동 저장           활성     │
│                          (연동 시)   │
├──────────────────────────────────────┤
│ ❤️ 심박수 모니터링     Apple Watch 필요│
└──────────────────────────────────────┘
```
- HealthKit 권한 요청 버튼
- 연동 상태 표시 (초록: 연동됨)
- 운동 자동 저장: HealthKit 연동 시 자동 활성
- 심박수: Apple Watch 필요 (워치 없으면 비활성)

### 6. 화면 섹션

```
┌──────────────────────────────────────┐
│ 🌓 외관 모드    [시스템|라이트|다크]  │
│                 (Segmented Picker)    │
├──────────────────────────────────────┤
│ ℹ️ 버전 정보              1.0.0     │
└──────────────────────────────────────┘
```

#### 외관 모드
| 모드 | 아이콘 | 동작 |
|------|--------|------|
| 시스템 | `circle.lefthalf.filled` | 시스템 설정 따름 |
| 라이트 | `sun.max.fill` | 항상 라이트 모드 |
| 다크 | `moon.fill` | 항상 다크 모드 |

| 서비스 | 용도 |
|--------|------|

| `HealthKitService` | HealthKit 권한 요청/상태 |
