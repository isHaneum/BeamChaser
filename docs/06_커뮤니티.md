# 커뮤니티 화면 (CommunityView)

## 파일 위치
`BeamChaser/Views/Community/CommunityView.swift`

## 화면 설명
세 번째 탭으로 표시되는 커뮤니티 화면입니다. "러닝 메이트" 모집과 "피드" 두 개의 서브 탭으로 구성됩니다.

---

## 네비게이션 제목
"커뮤니티" (large 스타일)

## 네비게이션 바
- **우측**: `+` 원형 아이콘 → 현재 탭에 맞는 새 글 작성 시트

---

## 화면 구성

### 탭 선택기
```
┌─────────────────────────────────┐
│  [👥 러닝 메이트]  [💬 피드]   │
└─────────────────────────────────┘
```
- 커스텀 탭 버튼 2개 (segmented 스타일)
- 선택됨: 흰색 텍스트 + accent 배경
- 미선택: 연한 텍스트 + 카드 배경

---

## 1. 러닝 메이트 탭

### 지도 프리뷰
```
┌──────────────────────────────────────┐
│                                      │
│   📍 반포한강공원                     │
│          📍 여의도공원                │
│   📍 잠실운동장                      │
│                                      │
└──────────────────────────────────────┘
```
- 모집글의 위치를 지도에 핀으로 표시
- 검색 바: 장소 검색 기능 (MKLocalSearch)
- 높이: 200pt

### "내 근처 러닝 모집" 섹션 헤더
- 위치 아이콘 + 텍스트

### 모집 카드
```
┌──────────────────────────────────────┐
│ 🏃 지민 (골드)                       │
│ 한강 반포 주말 러닝 같이 하실 분!     │
│                                      │
│ 📍 반포한강공원                      │
│ 📅 3/30 (일) 07:00                  │
│ 🏎 5'30"/km  │  📏 10km            │
│                                      │
│ 편하게 대화하면서 달려요.              │
│ 초보도 환영합니다!                    │
│                                      │
│ 👥 3/6명              [참가하기]     │
│                      2시간 전        │
└──────────────────────────────────────┘
```

- **작성자**: 이름 + 등급 뱃지 (등급별 아이콘)
- **제목**: 볼드 텍스트
- **장소**: 위치 핀 아이콘
- **일시**: 날짜 + 시간
- **페이스/거리**: 목표 페이스, 목표 거리
- **설명**: 자유 텍스트
- **참가**: 현재 인원/최대 인원 + 참가 버튼
  - 참가 시: 인원 +1, 버튼 → "참가 취소" (빨강)
  - 정원 초과 시: 참가 불가
- **시간**: 작성 시각 (상대 시간)

### 모집 작성 시트 (CreateMatePostView)
- 제목, 장소, 날짜, 시간, 목표 페이스, 목표 거리, 최대 인원, 설명 입력
- 장소: 지도에서 선택 (MKLocalSearch 연동)

## 2. 피드 탭

### 피드 게시글 카드
```
┌──────────────────────────────────────┐
│ 지민 (골드)                  30분 전 │
│                                      │
│ 오늘 한강에서 10K 완주!              │
│ BeamChaser 레이저 덕분에 페이스 유지 성공│
│                                      │
│ 📏 10.2 km   🏎 5'24"/km           │
│                                      │
│ ❤️ 12   💬 2              [좋아요]  │
│                                      │
│ 현우 (실버): 축하합니다! 대단해요    │
│ 소연 (레이저): 다음엔 같이 달려요!   │
│                                      │
│ 댓글 입력...                [전송]   │
└──────────────────────────────────────┘
```

#### 게시글 타입
| 타입 | 설명 |
|------|------|
| `runResult` | 러닝 결과 공유 (거리 + 페이스 표시) |
| `mateFinding` | 러닝 메이트 찾기 |
| `freeBoard` | 자유 게시판 (거리/페이스 없음) |

#### 기능
- **좋아요**: 하트 아이콘 탭 → 토글 (빨강/회색)
- **댓글**: 인라인 댓글 입력 + 전송 버튼
- **사진**: 게시글에 사진 첨부 가능 (PhotosPicker 연동)
- **작성자 정보**: 이름 + 러너 등급 아이콘/색상

### 피드 작성 시트 (CreateFeedPostView)
- 내용 텍스트 입력 (TextEditor)
- 사진 첨부 (PhotosPicker)
- 게시글 타입 선택 (결과 공유 / 자유)
- 최근 러닝 기록 자동 연결 가능

## 데이터 모델

### RunnerPost
```swift
struct RunnerPost: Identifiable {
    let authorName: String
    let authorLevel: RunnerLevel
    let content: String
    let distanceKm: Double?
    let paceFormatted: String?
    let timeAgo: String
    var likes: Int
    var comments: [PostComment]
    let type: PostType     // .runResult / .mateFinding / .freeBoard
    var isLiked: Bool
    var photoData: Data?
}
```

### RunMatePost
```swift
struct RunMatePost: Identifiable {
    let authorName: String
    let authorLevel: RunnerLevel
    let title: String
    let location: String
    let coordinate: CLLocationCoordinate2D
    let date: String
    let time: String
    let targetPace: String
    let targetDistance: String
    var currentMembers: Int
    let maxMembers: Int
    let description: String
    var isJoined: Bool
}
```

## ViewModel: CommunityViewModel

| 메서드 | 기능 |
|--------|------|
| `toggleLike(postId:)` | 좋아요 토글 (+1 / -1) |
| `addComment(postId:, content:)` | 댓글 추가 |
| `toggleJoin(postId:)` | 모집 참가/취소 |
| `addMatePost(_:)` | 모집글 추가 |
| `addFeedPost(_:)` | 피드글 추가 |

---

## 샘플 데이터
현재는 로컬 샘플 데이터로 동작합니다 (서버 미연동):
- 러닝 메이트 모집 3건 (반포한강공원, 여의도공원, 잠실종합운동장)
- 피드 게시글 3건 (러닝 결과 2건, 자유 게시판 1건)
- 댓글 샘플 포함
