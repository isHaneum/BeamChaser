# 러닝 기능 (PaceSetupView + RunActiveView + RunMapView)

## 파일 위치
- `BeamChaser/Views/Running/PaceSetupView.swift` — 러닝 설정
- `BeamChaser/Views/Running/RunActiveView.swift` — 러닝 중 화면
- `BeamChaser/Views/Running/RunMapView.swift` — 러닝 지도

---

## 1. 러닝 설정 화면 (PaceSetupView)

### 화면 설명
러닝 시작 전 목표를 설정하는 화면입니다. 일반 모드와 인터벌 모드 두 가지를 지원합니다.

### 화면 구성

#### 탭 선택 (상단)
```
┌─────────────────────────┐
│  [일반]  |  [인터벌]    │  ← Segmented Picker
└─────────────────────────┘
```

#### 일반 모드 — 목표 토글 박스 3개
```
┌─────────┬─────────┬─────────┐
│  🏁     │  ⏱     │  🏎     │
│  5.0    │  30     │  5'30"  │
│  km     │  분     │  /km    │
│  거리   │  시간   │  페이스  │
└─────────┴─────────┴─────────┘
  [OFF]     [OFF]     [OFF]
```

- 3개 정사각형 박스가 화면 1/3씩 차지
- 각 박스를 탭하면:
  - 비활성 → 활성화 + 설정 패널 펼침
  - 활성 + 동일 박스 → 설정 패널 접기
  - 활성 + 다른 박스 → 해당 박스 패널로 전환
- OFF 버튼으로 개별 비활성화 가능
- **독립적으로 여러 목표 동시 설정 가능** (거리 + 페이스 등)

#### 거리 설정 패널
```
┌──────────────────────────┐
│         5.0 km           │
│  [3.0] [5.0] [10.0] [21.1] │ ← 프리셋
│  ━━━━━━━●━━━━━━━ (슬라이더) │ ← 0.5~42.2km
└──────────────────────────┘
```

#### 시간 설정 패널
```
┌──────────────────────────┐
│         30 분             │
│  [15] [30] [45] [60]     │ ← 프리셋
│  ━━━━━●━━━━━ (슬라이더)   │ ← 5~180분
└──────────────────────────┘
```

#### 페이스 설정 패널
```
┌──────────────────────────┐
│      5'30" /km           │
│  [초급7:00] [중급5:30]   │
│  [고급4:30] [엘리트3:30] │ ← 프리셋
│                          │
│   분: [◀ 5 ▶]            │ ← +/- 버튼
│   초: [0][10][20][30]... │ ← 선택 칩
└──────────────────────────┘
```

#### 인터벌 모드
- 프리셋 3종 제공:
  - "5km 인터벌" (5구간)
  - "웜업-본운동-쿨다운" (3구간)
  - "피라미드 인터벌" (7구간)
- 커스텀 인터벌 생성 가능 (구간명, 거리, 페이스 설정)
- 각 구간을 카드로 표시 (색상 구분)

#### 플로팅 시작 버튼
```
┌─────────────────────────┐
│  🏃  러닝 시작           │  ← 하단 고정, 오렌지 그라데이션
└─────────────────────────┘
```

## 2. 러닝 중 화면 (RunActiveView)

### 화면 설명
러닝이 진행 중인 전체 화면입니다. 지도 위에 실시간 데이터가 오버레이됩니다. 네비게이션 바가 숨겨져 완전한 풀스크린으로 동작합니다.

### 화면 구성

#### 상단 오버레이
```
┌────────────────────────────────────┐
│ ▐▐▐ 강함  [본운동 5'00"]  ⚡72%  │  ← GPS 신호 / 인터벌 구간 / 목표 진행률
│                                    │
│ ↑ 앞서는 중              🔴 +15m │  ← 페이스 상태 배너 + 레이저 갭
│ ━━━━━●━━━━━━━🔴━━━━━━━━━━━━━━━  │  ← 레이저 갭 바 (미니)
└────────────────────────────────────┘
```

##### GPS 신호 표시
- 3단계 바: 강함(3칸, 초록) / 보통(2칸, 노랑) / 약함(1칸, 빨강) / 없음(0칸)
- 수평 정확도 기반: <10m 강함, <20m 보통, ≥20m 약함

##### 페이스 상태 배너
- **앞서는 중** (ahead): 초록 배경 + ↑ 아이콘
- **페이스 유지** (onPace): 오렌지 배경 + = 아이콘
- **뒤처지는 중** (behind): 빨강 배경 + ↓ 아이콘
- 우측에 레이저 갭 숫자 크게 표시 (28pt 메트릭 폰트)

##### 레이저 갭 바
- 중앙 = 목표 위치 (오렌지 점)
- 레이저 도트가 갭에 따라 좌우로 이동
- 최대 범위: ±100m

##### 인터벌 구간 표시
- 인터벌 모드일 때만 표시
- 구간 이름 + 목표 페이스
- 색상 구분: <4:30 빨강, <5:30 오렌지, ≥5:30 초록

##### 목표 진행률 배지
- 거리/시간 목표가 있을 때 우측 상단에 퍼센트 표시
- 100% 달성 시 초록으로 변경

#### 하단 오버레이

##### 간결 메트릭 (기본)
```
┌────────────────────────────────┐
│          ━━━ (핸들)            │
│  2.45     12:34     5'02"     │
│   km      시간      /km      │
│                               │
│    [🔒]    [⏸ 일시정지]       │
└────────────────────────────────┘
```
- 거리(km) / 시간 / 페이스(/km) 3칸 표시
- 심박수 모니터링 시 4번째 칸 추가 (bpm)

##### 확장 메트릭 (탭하여 펼침)
```
┌────────────────────────────────┐
│          ━━━ (핸들)            │
│         2.45 km               │ ← 대형 거리 (56pt)
│  12:34   5'02"/km   5'30"    │ ← 시간 / 현재 페이스 / 목표
│  🔴 레이저 갭  +15m  +3초    │ ← 갭 상세
│  ❤️ 심박수    145 bpm 320kcal│ ← 심박수 (워치 연동 시)
│  ⌚ 워치 페이스  5'00"/km     │ ← Apple Watch 데이터
│                               │
│    [🔒]    [⏸ 일시정지]       │
└────────────────────────────────┘
```

### 컨트롤 버튼 (3가지 상태)

#### 러닝 중
```
[🔒 잠금]    [⏸ 일시정지]    (빈 공간)
 (52pt)        (72pt 메인)
```

#### 일시정지 중
```
         ● 일시정지 (깜빡임)
   [⏹ 종료]        [▶ 계속 달리기]
   (빨강, 64pt)      (초록, 80pt)
```

#### 러닝 완료
```
┌────────────────────────────────┐
│  ✅ 러닝 완료                  │
│  2.45 km   12:34   5'02"/km  │
│                               │
│  [✓ 완료]  ← 오렌지 캡슐 버튼 │
└────────────────────────────────┘
```

---

### 목표 달성 알럿
- 설정한 거리/시간 목표 달성 시 알럿 표시
- "계속 달리기" 또는 "종료하기" 선택

### 종료 확인 알럿
- "러닝을 종료할까요?"
- "계속 달리기" / "종료" 선택

## 3. 러닝 지도 (RunMapView)

### 화면 설명
RunActiveView에 임베드되는 지도 뷰입니다.

### 지도 요소
| 요소 | 설명 |
|------|------|
| 현재 위치 | 파란 원 (실기기: UserAnnotation) |
| 러닝 경로 | 네온 그린 라인 (글로우 배경 14pt + 메인 5pt) |
| 시작점 | 초록 원 (10pt) |
| 레이저 위치 | 빨간 글로우 도트 (LaserDot, 16pt) |

### 레이저(가상 주자) 위치 계산
- 목표 페이스 기반으로 갭 거리만큼 이동 방향 앞/뒤에 표시
- 이동 방향: `course` 값 또는 최근 두 좌표의 atan2로 계산
- 위도/경도 변환: 111,320m/도 기준

---

## BLE 연동 (러닝 중)
- `onAppear`: 장치 연결 시 `startRun()` 명령 + 목표 페이스 전송
- `onDisappear`: 비정상 종료 시 `stopRun()` 명령
- 거리 변경 시: `sendTargetPace()` 호출

---

## 의존하는 서비스
| 서비스 | 용도 |
|--------|------|
| `RunSessionManager` | 러닝 상태, 타이머, 기록 저장 |
| `LocationService` | GPS 위치, 거리, 속도, 경로 |
| `BLEService` | 장치 명령 전송 |
| `HealthKitService` | 심박수, 칼로리, 워치 페이스, 운동 기록 |
| `PaceMakerEngine` | 갭 계산, 페이스 상태 판정 |

```
HomeView → PaceSetupView → RunActiveView
                               ↓ (완료)
```
