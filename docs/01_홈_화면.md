# 홈 화면 (HomeView)

## 파일 위치
`BeamChaser/Views/Home/HomeView.swift`

## 화면 설명
앱 실행 시 첫 번째 탭으로 표시되는 메인 화면입니다. 전체 화면에 지도가 표시되고, 하단에 드래그 가능한 패널이 있어 러닝을 시작할 수 있습니다.

---

## 화면 구성

### 1. 전체 화면 지도
- **MapKit** 기반 지도가 화면 전체를 차지합니다
- 유저의 현재 위치가 파란 원으로 표시됩니다
- 지도 스타일: `.standard(pointsOfInterest: .excludingAll)` — POI 제거하여 깔끔한 지도
- 시뮬레이터에서는 서울시청 좌표(37.5665, 126.9780)가 기본 위치로 표시됩니다

### 2. 현위치 버튼 (우측 상단)
- 원형 버튼으로 위치 아이콘(`location.fill`) 표시
- 탭하면 지도가 현재 위치로 이동 (500m 범위)
- `.ultraThinMaterial` 배경에 그림자 적용

### 3. 하단 패널 (드래그 가능)
#### 펼친 상태 (기본)
```
┌─────────────────────────────┐
│         ━━━ (핸들)          │
│                             │
│  RUN BEAM          [장치칩] │
│  라인 레이저 페이스메이커     │
│                             │
│  ┌─────────────────────┐    │
│  │  ▶  시작하기         │    │
│  └─────────────────────┘    │
└─────────────────────────────┘
```

- **로고**: "RUN BEAM" (hero 폰트 24pt) + "라인 레이저 페이스메이커" 서브 타이틀
- **장치 상태 칩**: BLE 연결 상태를 초록(연결)/빨강(미연결) 원으로 표시, 탭 시 `DeviceConnectionView`로 이동
- **시작 버튼**: 오렌지 그라데이션 캡슐 버튼, 탭 시 `PaceSetupView`로 네비게이션

#### 접힌 상태
```
┌─────────────────────────────┐
│  RUN BEAM             ▲    │
│         ━━━ (핸들)          │
└─────────────────────────────┘
```
- 로고와 핸들만 표시
- 지도를 더 넓게 볼 수 있음

#### 드래그 제스처
- 아래로 60px 이상 드래그 → 접힘
- 위로 60px 이상 드래그 → 펼침
- 스프링 애니메이션 적용 (response: 0.35, damping: 0.8)

---

## 장치 미연결 경고 (Alert)
시작 버튼을 눌렀을 때 BLE 장치가 연결되어 있지 않으면 알림이 표시됩니다.

- **제목**: "BeamChaser 장치 미연결"
- **메시지**: "BeamChaser 장치가 연결되지 않았습니다. 레이저 페이스 가이드 없이 앱 단독 러닝 모드로 시작할 수 있습니다."
- **버튼 1**: "장치 연결하기" → BLE 스캔 시작
- **버튼 2**: "단독 모드로 시작" → `PaceSetupView`로 이동

---

## 위치 권한 요청
- `onAppear`에서 `locationService.requestPermission()` 호출
- 위치를 받으면 지도가 500m 범위로 자동 줌
- 실기기에서만 동작 (시뮬레이터에서는 GPS 요청 생략)

---

## 네비게이션 흐름
```
HomeView
├── NavigationLink → DeviceConnectionView (장치 칩 탭)
└── NavigationDestination → PaceSetupView (시작 버튼 탭)
```

---

## 의존하는 서비스
| 서비스 | 용도 |
|--------|------|
| `BLEService` | 장치 연결 상태 확인 (`isConnected`) |
| `LocationService` | 현재 위치, 권한 요청 |

---

## 디자인 특징
- 배경: `.ultraThinMaterial` (다크 모드 환경)
- 패널 모서리: 28pt 라운드
- 그림자: 패널 상단에 `.black.opacity(0.4)` 그림자
- 시작 버튼: `RBColor.accentGradient` (오렌지 → 다크 오렌지)
