# 🏃‍♂️ Beam Chaser - watchOS Companion App

## 🎯 1. Project Overview (프로젝트 개요)
Beam Chaser watchOS 앱은 물리적 레이저 페이스메이커 하드웨어와 연동되는 궁극의 러닝 컨트롤러입니다. 아이폰을 꺼내지 않고도 워치만으로 기기를 제어하며, 심박수와 페이스 격차를 실시간으로 모니터링합니다. 
* **목표:** 기존 NRC나 애플 운동 앱을 대체할 수 있는, **'시각(Neon UI)'과 '촉각(Haptics)'** 중심의 혁신적인 러닝 경험 제공.

---

## 2. Core UX/UI Philosophy (디자인 철학)
기존의 숫자를 읽는 딱딱한 방식에서 벗어나, **'직관적인 빛의 감성'**과 **'촉각'**을 중심으로 유저가 상황을 원초적으로 느낄 수 있도록 디자인합니다. '네온 레드 on 트루 블랙' 테마를 더욱 역동적으로 적용합니다.

1. **Glanceability (0.1초 인지):** 달리는 중 흔들리는 손목을 슬쩍 보는 것만으로도 상황을 인지. 색상과 게이지 바(Bar) 위주로 디자인.
2. **Neon Red & True Black:** 완전한 검은색(True Black) 배경에 네온 레드 포인트. OLED 배터리 절약 + 야간 러닝 강렬한 인상.
3. **Eyes-Free Control:** 디지털 크라운으로 노룩 페이스/밝기 조절.

---

## 3. Screen Flow & Design Details (화면 구성 및 디자인)

### 3.1. Quick Start Screen (앱 실행 즉시 화면)
사용자가 앱을 켜자마자 가장 먼저 보게 되는 화면. 즉각적인 러닝 시작 지원.

- 완전한 True Black 배경
- 상단: 시간(좌) / 심박수 아이콘+수치(네온 레드) + GPS 도트(우)
- 중앙: 거대한 원형 네온 레드 버튼. 버튼 내부에 달리는 사람 아이콘 + "QUICK RUN" 텍스트
- 하단: 현재 목표 페이스(네온 레드 대형) + "크라운으로 목표 페이스 조절" 안내
- 디지털 크라운: 목표 페이스 3:00~12:00/km 조절 (5초 단위, 햅틱)

### 3.2. Main Dashboard (러닝 중 화면)
- True Black 배경
- 상단: 시간 / 심박수(네온 레드) / GPS 도트
- 중앙 (The Chaser Gauge): 원형 네온 레드/그린 격차 게이지
  - 뒤처짐: 네온 레드 아크 + 글로우, "뒤처지는 중"
  - 앞섬: 네온 그린 아크 + 글로우, "앞서는 중"
  - 일치: 오렌지 아크, "ON PACE"
  - 게이지 중앙에 격차 수치 거대하게 표시 (예: +2, 아래 작게 m)
- 하단: 2x2 데이터 그리드 [페이스][시간][거리][속도] - 네온 레드 텍스트

### 3.3. Hardware Control (기기 제어 화면)
- True Black 배경
- 상단: 시간 / 심박 아이콘 / 장치 배터리 %
- 중앙: 수평 네온 레드 레이저 밝기 슬라이더 바. 크라운으로 제어. 노브에 크라운 아이콘 표시.
- 하단: "레이저 투사 영점 조절" + 네온 레드 테두리 큰 아래/위 버튼 + 각도 수치

### 3.4. Pause & Finish (종료 화면)
- True Black 배경
- 상단: 시간 + 상태 텍스트
- 중앙 (재개 버튼): 네온 그린(또는 오렌지) 대형 버튼 (height 46)
- 하단 (종료 버튼): 네온 레드 버튼 (height 36)
- 최하단: 2x2 요약 그리드 (거리 | 시간 | 페이스 | 칼로리)

## 📳 4. Haptic Feedback System (독자적인 진동 피드백)
워치의 가장 강력한 무기인 햅틱 엔진을 레이저 격차 안내에 적극 활용합니다. 이어폰이 없어도 몸으로 페이스를 느낄 수 있게 합니다.

* **페이스 뒤처짐 (Catch up!):** 짧고 날카로운 진동 2회 (탁! 탁!)
* **페이스 초과 (Slow down):** 길고 묵직한 진동 1회 (우웅-)
* **목표 일치 (On pace):** 1km 지날 때마다 또는 인터벌 구간 변경 시 심장 박동 같은 부드러운 진동 (두근- 두근-)

## 🛠 5. Technical Requirements (기술적 요구사항)
* **Framework:** `SwiftUI` (워치 UI 전용), `WatchConnectivity` (아이폰과의 실시간 데이터 및 명령 동기화)
* **HealthKit:** `HKWorkoutSession`을 사용하여 백그라운드 구동, 심박수 측정 및 활동 링(Activity Rings) 연동.
* **CoreBluetooth:** 워치 단독으로 레이저 기기와 직접 연결(Standalone mode)할 것인지, 아이폰을 거쳐 통신할 것인지 아키텍처 결정 필요.
